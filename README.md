# Afzal Khan — Portfolio

A modern, premium portfolio website for **Afzal Khan** — Technical Manager specializing in BIM, GIS, 3D Laser Scanning, Digital Twins, and Geospatial Solutions based in Abu Dhabi, UAE.

## ✨ Features

- 🎨 Premium dark theme with glassmorphism & cyan accents
- 🖱️ Cursor glow, tilt effects, magnetic buttons
- 📱 Fully responsive (mobile-first)
- ⚡ Fast loading — pure HTML/CSS/JS, no dependencies
- 🔍 SEO optimized with meta tags & semantic HTML
- ♿ Accessible with ARIA labels
- 🎭 Smooth scroll reveal animations
- 📊 Animated skill bars & career timeline

## 🧩 Sections

1. **Hero** — Bold headline, profile photo, floating stats
2. **About** — Bio, highlight cards, tech stack tags
3. **Skills** — 3 categories with animated progress bars
4. **Experience** — Interactive timeline (7 positions)
5. **Projects** — Card grid with hover tilt effects
6. **Certifications** — Education & credentials
7. **Contact** — Form + email/LinkedIn/location
8. **Footer** — Minimal with social links

## 🚀 Quick Start

Just open `index.html` in your browser — no build step needed!

```bash
# Or use a local server
python -m http.server 8000
```

## 🌐 Deploy

Drag the entire folder into [Netlify Drop](https://app.netlify.com/drop) for instant deployment.
